
✅ Add it as metadata (visible via tools like exiftool)

😈 Hide it inside the image (steganography)

Let’s break both down.

✅ Method 1 — Add a Message as Metadata (Clean & Simple)
Use exiftool.

Add a simple message:
exiftool -Comment="Hi" image.jpg
Or a longer message:

exiftool -ImageDescription="This is a secret test message" image.jpg
If you don’t want a backup file:

exiftool -overwrite_original -Comment="Hi" image.jpg
Check it:
exiftool image.jpg
You’ll see your message in:

Comment

ImageDescription

UserComment (depending what you used)

🔥 Pro Trick — Use Custom Fields
exiftool -UserComment="Hidden message here" image.jpg
😈 Method 2 — Hide a Message Inside the Image (Steganography)
Now we enter real cybersecurity territory.

In Kali, use steghide.

Install (if needed)
sudo apt install steghide
Hide a Message
First create a text file:

echo "Hi this is secret" > secret.txt
Now embed it:

steghide embed -cf image.jpg -ef secret.txt
It will ask you to set a password.

Now your image visually looks the same — but contains hidden data.

Extract It Later
steghide extract -sf image.jpg
Enter password → it gives back secret.txt
