When you say “delete complete metadata”, you’re talking about stripping everything — EXIF, IPTC, XMP, GPS, camera info, thumbnails, all of it.

Here’s the clean way to do it with exiftool.

🧹 Remove ALL metadata from an image
exiftool -all= image.jpg
That’s it.

What this does:
-all= → deletes every metadata tag

Creates a backup automatically (image.jpg_original)

🚫 Remove metadata WITHOUT creating a backup
By default, exiftool makes a backup file. If you don’t want that:

exiftool -all= -overwrite_original image.jpg
Now it edits the file in place.

🗂 Remove metadata from multiple images
exiftool -all= -overwrite_original *.jpg
Or entire directory:

exiftool -all= -overwrite_original -r /path/to/folder
-r = recursive (subfolders included)

🔍 Verify metadata is gone
Run:

exiftool image.jpg
If you see almost nothing except basic file info (like file size), you're good.
